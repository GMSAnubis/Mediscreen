Welcome to Mediscreen

When you writes notes about your patients keep in mind that these words are use as triggers to calculate diabetes risks. They are only counted once, even if you write them multiple times, in order to avoid false results.

For example if you write "patient is below height" or "patient height is over normal height", the trigger word height is counted only once, regardless of the number of times this word is writtent, whetether une one or multiple notes. Triggers words are not case sensitive

Triggers words:
  • Hemoglobin A1C
  • MicroAlbumin
  • Height
  • Weight
  • Smoker
  • Abnormal
  • Cholesterol
  • Dizziness
  • Relapse
  • Reaction
  • Antibodies

If you want to add more triggers words, or modify the risk calculation process, feel free to contact us.
